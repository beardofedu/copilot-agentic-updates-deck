Topic 7

Copilot CLI — Terminal-Native Dev Agent

  • GitHub-native agent in your terminal: works directly with issues, branches, and pull requests via native /mcp support.
  • /fleet runs parallelized subagents across multiple models simultaneously; /plan + autopilot mode carries tasks forward without step-by-step approval.
  • /resume keeps long-running sessions alive with persistent memory and context compaction.
  • Extend with AGENTS.md, Agent Skills, and custom MCP servers; /delegate a branch-to-PR change in one command.
  • Included in all Copilot plans (Free, Pro, Business, Enterprise); inherits existing org governance policies automatically.

Source: GitHub — Copilot CLI

Topic 8

Work IQ — M365 Copilot Intelligence Layer

  • Always-on intelligence fabric that semantically understands organizational data across Outlook, Teams, OneDrive, and SharePoint.
  • Delivers pre-assembled, permission-aware work context to agents — no custom data pipelines required.
  • Supports agent-to-agent (A2A) collaboration and multi-step workflows via REST and MCP APIs.
  • Graph Connectors and third-party plugins bring external LOB data and actions directly into Copilot reasoning.
  • Centralized governance, sensitivity labels, and compliance controls managed from the M365 admin center.

Source: Microsoft Learn — Work IQ overview
